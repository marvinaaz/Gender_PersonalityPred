The transcriptions corresponding to small groups recorded in English are available in the folder
	txtFiles

The respective id of the speaker and the letter in the group are listed in the file
	LettersandIdSpeaker.csv


For analysis, we generated a single file per participant (and removed indications of Laught, Crosstalk, and labels from timing) to extract the Word Categories  using LIWC. The word categories are in the folder
	LIWC_categories
In this folder the resultant word categories are available in two formats:
	csv and matlab file


Refer to paper MUM'12 for more details.
Assessing the Impact of Language Style on Emergent Leadership Perception from Ubiquitous Audio, Dairazalia Sanchez-Cortes, Petr Motlicek and Daniel Gatica-Perez, in: Proceedings of the 11th International Conference on Mobile and Ubiquitous Multimedia, Ulm, Germany, 2012
